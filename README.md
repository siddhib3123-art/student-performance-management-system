# student-performance-management-system
Django web application to manage student academic records with CRUD operations, authentication, search, and automatic grade calculation
## Features
- Add student details
- Update student information
- Delete student records
- Calculate student grades
- View student performance

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run the Project

1. Clone the repository
2. Install Django
3. Run the server
